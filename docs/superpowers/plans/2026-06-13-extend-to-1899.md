# Plan: extend the long-count record back to 1899

**Goal.** Push the project's spine — the *share of the final certified vote
counted by election night* — back from its current 1960 floor to **1899** (the
earliest row in the DOE turnout table), turning "sixty-plus years" into a
~125-year record. Reinforces the thesis by lengthening the flat, fast-count
baseline against which the modern VBM-driven slowdown reads.

**Core argument (user-confirmed).** Absentee voting was largely not practiced
before 1964, so the **in-person count ≈ total ballots cast ≈ certified final** —
which means election night delivered essentially the whole count (a ~100% night
floor). We do *not* need a per-election precinct/absentee split for this; it
follows from absentee being negligible. The newspaper archive's role is the
**cross-check that the count was actually known night-of** (i.e. counted fast
enough), not merely that it theoretically could have been.

**Date:** 2026-06-13. **Branch:** new feature branch off `main`.

---

## What the repo establishes (don't re-derive)

1. The deliverable is the **night-share floor**, not raw vote totals. See
   `docs/analysis/2026-06-09-counting-speed-trend.md` Finding #2: "arithmetic
   alone proves election night once delivered nearly the whole count."
2. **Denominators back to 1899 already exist** in
   `data/sf_turnout_history_doe_1899_2019.csv` (registration + ballots_cast for
   1899, 1902–03, 1908, 1910, 1912…1956 then 1960). These are the certified
   totals every floor divides into — the work to acquire them is already done.
3. The **in-person "diamond" floor** (precinct ballots ÷ certified total) is a
   guaranteed night floor needing *no newspaper*. The `precinct`/`mail` columns
   are `n/a` before 1964 — but per the core argument, with absentee ≈ 0 the
   in-person count ≈ the final, so the floor is ~100% without needing the split.
4. The **NewsBank image edition runs 1865–2017** (runbook, "Where the data
   lives") — so 1899–1959 morning-after Chronicle results boxes are reachable
   with the *existing* capture/triage/read pipeline in `scripts/archive-recovery/`.
5. Schema already supports this: a pre-1960 row drops straight into
   `data/sf_archival_canvass_points.csv` (`extraction=newsbank-image-scan`,
   `stamp_kind=news-derived`) or, for floor-only rows, into the night-floor
   table. **No new file formats required.**
6. Hazards that carry backward: the denominator-error rule
   (`docs/denominator-errors.md`), masthead-date verification, the **1906
   earthquake** records gap (no 1904/1906 rows), and the **1911 women's
   suffrage** registration jump (1910→1912 ballots cast nearly doubles).

## Approach (revised after Phase-2 findings — see analysis doc)

The "structural floor" idea (a flat in-person bound) is **abandoned**: the first
two newspaper cross-checks showed pre-1964 night-of completeness is NOT flat —
1916 SF was ~44% of precincts in at press time, 1932 SF was complete. So there
is no defensible single floor to assume; **each election's night-of share must
be measured from the morning-after paper.**

**Decision (user, 2026-06-13): full pull, not a sample.** Recover the real
night-of count for *every* pre-1964 election with a DOE denominator (29; 1916 &
1932 done), let the trend emerge from the data. The story is now three acts:
slow hand-count era → fast machine/punch-card interlude → modern mail slowdown.

## Phase 0 — Feasibility & the gating fact (desk research, no login)

The structural floor rests on one era-level fact: **absentee voting was a
negligible share of the SF vote, 1899–1962.** Establish it with citations.

- [ ] Document CA/SF absentee-voting prevalence pre-1964. CA civilian absentee
      voting was tightly restricted (excuse-required, in-person-witnessed)
      until the **1962** liberalization; statewide absentee share sat in the low
      single digits through this era. Cite: CA Secretary of State election-law
      history + any SoS **Statement of Vote** absentee lines that exist for the
      period (archive.org / California State Library).
- [ ] **Where available**, pull the actual in-person/absentee split per election
      from the SoS Statement of Vote or SF Registrar annual reports — a real
      figure beats an era assumption and matches the 1964+ rows' provenance.
- [ ] Inventory the **target election set**: every row already in the DOE table
      1899–1956 (~25 elections), flagged real-split vs. era-assumption. Note the
      known gaps (1904, 1906 — quake; odd municipal years).
- [ ] Decide **floor labelling** for honesty on `/sources` and `/methods`:
      - real split → `method = non-absentee (precinct) share of certified total`
        (matches existing 1964+ rows);
      - era assumption → `method = in-person floor, absentee ≤ N% (era, <cite>)`.

Exit criterion: a cited statement that absentee ≤ N% for the era, plus the
target-election list each with its certified total.

## Risks & open questions

- **The negligible-absentee claim is the linchpin.** Phase 0 must cite it. If
  absentee turns out non-trivial for some sub-era, those elections need the
  real split or get *no* floor (omit, don't guess) and land on `/missing`.
- **"Counted night-of" is an assumption until Phase 2 checks it.** Pre-1964
  hand-counts of paper ballots could run past election night for down-ballot
  races; the floor is about ballots *cast in person*, and the newspaper check
  confirms how much was actually *reported* by morning. Treat a lagging count
  as a finding, not an error.
- **Sparse, irregular pre-1960 calendar.** The DOE table skips odd years and the
  quake years; this is points-on-a-trend, not a dense series. Frame it that way
  in copy (the modern data is already "n is small everywhere").
- **`pct_of_final` > 100 risk.** Pre-1960 floors should sit just under 100%;
  any value above it signals a denominator error — route to
  `denominator-errors.md`, don't clamp silently.
- **Examiner is microfilm-only**; rely on the Chronicle image edition for
  Phase 2, consistent with current coverage.

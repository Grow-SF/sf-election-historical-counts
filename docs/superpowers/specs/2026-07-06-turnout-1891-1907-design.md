# Turnout 1891-1907: fill the registration/ballots gap

**Date:** 2026-07-06
**Status:** approved in conversation (follow-up to the night-count recovery)
**Owner:** Claude session, operator Steven

## Goal

Fill the turnout chart's 1891-1907 hole: registration and ballots-cast for
the sixteen SF elections between the mined FY1888-89 Municipal Reports
Registrar table (ends 1890) and the reliable start of the DOE turnout table
(1908). Browserless: archive.org and HathiTrust volumes only.

## Worklist

Statewide even-year generals (municipal offices consolidated onto the same
ballot in this era): 1892-11-08, 1894-11-06, 1896-11-03, 1898-11-08,
1900-11-06, 1902-11-04, 1904-11-08, 1906-11-06.
Municipals/specials: 1897-12-27 (Freeholders), 1898-05-26 (charter
ratification), 1898-11-01, 1899-11-07, 1901-11-05, 1903-11-03, 1905-11-07,
1907-11-05. The DOE table's four sparse pre-1908 rows (1899-12-02,
1902-12-02, 1903-09-29, 1903-10-08) get cross-checked, not edited.

## Sources, in order of authority

1. **SF Municipal Reports** annual volumes (archive.org, the
   `sanfranciscomuni*sanfrich` series): the Registrar of Voters chapter
   prints registration and total votes cast per election, the same table
   that supplied 1879-1890 (`data/sf_turnout_pre1899.csv`).
2. **CA Statement of Vote / Blue Book / Governmental Roster** (archive.org,
   HathiTrust) for the statewide generals: county registration and vote
   totals; the proven vein of `data/pre1892_certified.md`.
3. Secondary cross-checks: Wikipedia by-county tables citing the State
   Archives Abstract of the Vote.

## Data flow

New source file `data/sf_turnout_1891_1907.csv` with columns
`election_date,registration,ballots_cast,basis,source` (basis:
`muni-registrar` | `sov-bluebook`; source: full citation with volume, page,
and archive.org/HathiTrust identifier). A small reader loop in
`scripts/build_viz_data.py` beside the pre1899 loop, tagging the JSON
`source` from `basis`. The DOE mirror CSV is never edited. REBUILD chain
and tests as in the night-count effort; the turnout README image reshoots
via the (now coverage-fitted) `scripts/shoot_charts.cjs`.

## Gates

- registration > ballots_cast; both plausible against the era curve
  (registration ~55k in 1890 rising to ~75k by 1908; the 1906 fire
  destroyed the Great Register, so a sharp 1906-1908 registration dip or
  re-registration artifact must be documented, not smoothed).
- Any top-contest sum available (e.g. from `pre1892_certified.md`-style
  sources) must be ≤ the ballots-cast figure.
- A row is only ingested with a genuine printed ballots-cast/total-vote
  figure AND a registration figure; contest-sum-only or registration-only
  findings are recorded in the source notes, not ingested.
- Load-bearing digits queue for the operator per the human-verification
  loop; conflicts with the DOE's sparse rows go to
  `docs/doe-data-discrepancies.md`.

## Out of scope

Night counts for these elections (that is the separate pre-1907 recovery
listed in the README); eligible-population denominators; any NewsBank work.
